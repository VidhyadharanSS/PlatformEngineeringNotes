### 9.3.2 HTTP Requests and REST APIs: Talking to Web Services

#### Why HTTP Clients Matter

Platform engineers constantly interact with APIs:
- **Kubernetes API** – Manage clusters programmatically
- **Cloud provider APIs** – AWS, GCP, Azure
- **Monitoring APIs** – Prometheus, Datadog
- **CI/CD APIs** – GitHub, GitLab, Jenkins
- **Webhooks** – Receive events from services

The `requests` library is the standard for HTTP in Python – simple, powerful, and widely used.

This note covers HTTP requests. Note 9.3.1 covered logging; note 9.3.3 is the subchapter review.

**Backward references:** HTTP from Module 2 (methods, status codes, headers); REST APIs from Module 8 (webhooks, API calls); JSON from Module 9 (parsing responses).

## Part 2: GET Requests – Fetching Data

### Basic GET Request

```python
import requests

# Simple GET
response = requests.get('https://api.github.com/users/octocat')

# Check status
print(f"Status code: {response.status_code}")
print(f"Success: {response.ok}")  # True if 200-399

# Get content as text
print(response.text)

# Parse JSON
data = response.json()
print(f"User: {data['login']}")
print(f"Name: {data['name']}")
print(f"Followers: {data['followers']}")
```

### GET with Parameters

```python
import requests

# Query parameters
params = {
    'q': 'requests python',
    'sort': 'stars',
    'order': 'desc',
    'per_page': 5
}

response = requests.get('https://api.github.com/search/repositories', params=params)
print(f"URL: {response.url}")  # https://api.github.com/search/repositories?q=requests+python&sort=stars&order=desc&per_page=5

data = response.json()
for repo in data['items']:
    print(f"- {repo['name']}: {repo['stargazers_count']} stars")
```

## Part 3: POST Requests – Creating Data

### POST with JSON

```python
import requests

# JSON data
data = {
    'title': 'New Issue',
    'body': 'This is the issue description',
    'labels': ['bug', 'urgent']
}

headers = {
    'Authorization': 'token ghp_xxx',
    'Content-Type': 'application/json'
}

response = requests.post(
    'https://api.github.com/repos/owner/repo/issues',
    json=data,
    headers=headers
)

if response.status_code == 201:
    print("Issue created!")
    print(f"URL: {response.json()['html_url']}")
else:
    print(f"Failed: {response.status_code}")
    print(response.text)
```

### POST with Form Data

```python
import requests

# Form data (application/x-www-form-urlencoded)

    'password': 'secret'
}
```

## Part 6: Handling Responses

### Checking Status Codes

```python
import requests

response = requests.get('https://api.github.com/users/octocat')

# Method 1: status_code
if response.status_code == 200:
    data = response.json()
elif response.status_code == 404:
    print("User not found")
elif response.status_code == 403:
    print("Rate limit exceeded")
else:
    print(f"Error: {response.status_code}")

# Method 2: ok (True for 200-399)
if response.ok:
    data = response.json()
else:
    print(f"Request failed: {response.status_code}")

# Method 3: raise_for_status()
try:
    response.raise_for_status()
    data = response.json()
except requests.exceptions.HTTPError as e:
    print(f"HTTP error: {e}")
```

### Response Methods

```python
response = requests.get('https://api.github.com/users/octocat')

# JSON response
data = response.json()

# Raw text
text = response.text

# Binary content
content = response.content

# Headers
print(response.headers['Content-Type'])
print(response.headers.get('X-RateLimit-Limit', 'Unknown'))

# Cookies
for cookie in response.cookies:
    print(f"{cookie.name}: {cookie.value}")

# Response time
print(f"Response time: {response.elapsed.total_seconds()} seconds")

# URL (after redirects)
print(f"Final URL: {response.url}")
```

## Part 7: Error Handling and Retries

### Exception Types

```python
import requests
from requests.exceptions import Timeout, ConnectionError, HTTPError, RequestException

try:
    response = requests.get('https://api.example.com/data', timeout=5)
    response.raise_for_status()
    data = response.json()
    
except Timeout:
    print("Request timed out")
except ConnectionError:
    print("Failed to connect")
except HTTPError as e:
    print(f"HTTP error: {e.response.status_code}")
except RequestException as e:
    print(f"Request failed: {e}")
```

### Retry Logic with Tenacity

```python
from tenacity import retry, stop_after_attempt, wait_exponential
import requests

@retry(
    stop=stop_after_attempt(3),
    wait=wait_exponential(multiplier=1, min=2, max=10)
)
def fetch_data(url):
    response = requests.get(url, timeout=5)
    response.raise_for_status()
    return response.json()

# Usage
try:
    data = fetch_data('https://api.example.com/data')
    print(data)
except Exception as e:
    print(f"All retries failed: {e}")
```

### Manual Retry with Backoff

```python
import time
import requests

def fetch_with_retry(url, max_retries=3, backoff_factor=2):
    for attempt in range(max_retries):
        try:
            response = requests.get(url, timeout=5)
            response.raise_for_status()
            return response.json()
        except requests.exceptions.RequestException as e:
            wait_time = backoff_factor ** attempt
            print(f"Attempt {attempt + 1} failed: {e}. Retrying in {wait_time}s...")
            time.sleep(wait_time)
    
    raise Exception(f"Failed after {max_retries} attempts")
```

## Part 8: Practical Examples for Platform Engineers

### Example 1: Kubernetes API Client

```python
import requests
import yaml
from pathlib import Path

class K8sClient:
    def __init__(self, kubeconfig_path=None):
        self.kubeconfig = self._load_kubeconfig(kubeconfig_path)
        self.server = self.kubeconfig['clusters'][0]['cluster']['server']
        self.token = self.kubeconfig['users'][0]['user']['token']
        self.ca_cert = self.kubeconfig['clusters'][0]['cluster'].get('certificate-authority-data')
        
        self.session = requests.Session()
        self.session.headers.update({'Authorization': f'Bearer {self.token}'})
    
    def _load_kubeconfig(self, path=None):
        path = path or Path.home() / '.kube/config'
        with open(path, 'r') as f:
            return yaml.safe_load(f)
    
    def get_pods(self, namespace='default'):
        url = f"{self.server}/api/v1/namespaces/{namespace}/pods"
        response = self.session.get(url, verify=False)  # In production, verify with CA
        response.raise_for_status()
        return response.json()
    
    def get_pod_logs(self, pod_name, namespace='default', tail=None):
        url = f"{self.server}/api/v1/namespaces/{namespace}/pods/{pod_name}/log"
        params = {'tailLines': tail} if tail else {}
        response = self.session.get(url, params=params, verify=False)
        response.raise_for_status()
        return response.text

# Usage
client = K8sClient()
pods = client.get_pods('production')
for pod in pods['items']:
    print(pod['metadata']['name'])
```

### Example 2: GitHub API – List PRs

```python
import requests
import os
from datetime import datetime

class GitHubClient:
    def __init__(self, token=None):
        self.token = token or os.environ.get('GITHUB_TOKEN')
        self.session = requests.Session()
        self.session.headers.update({
            'Authorization': f'token {self.token}',
            'Accept': 'application/vnd.github.v3+json'
        })
    
    def get_pull_requests(self, repo, state='open'):
        url = f'https://api.github.com/repos/{repo}/pulls'
        params = {'state': state, 'per_page': 30}
        
        response = self.session.get(url, params=params)
        response.raise_for_status()
        return response.json()
    
    def create_issue(self, repo, title, body, labels=None):
        url = f'https://api.github.com/repos/{repo}/issues'
        data = {
            'title': title,
            'body': body,
            'labels': labels or []
        }
        
        response = self.session.post(url, json=data)
        response.raise_for_status()
        return response.json()
    
    def add_comment(self, repo, issue_number, comment):
        url = f'https://api.github.com/repos/{repo}/issues/{issue_number}/comments'
        response = self.session.post(url, json={'body': comment})
        response.raise_for_status()
        return response.json()

# Usage
github = GitHubClient()
prs = github.get_pull_requests('octocat/Hello-World')
for pr in prs:
    print(f"#{pr['number']}: {pr['title']} by {pr['user']['login']}")
```

### Example 3: Prometheus Query

```python
import requests
from datetime import datetime, timedelta

class PrometheusClient:
    def __init__(self, url='http://localhost:9090'):
        self.url = url
    
    def query(self, query, time=None):
        """Instant query"""
        params = {'query': query}
        if time:
            params['time'] = time
        
        response = requests.get(f'{self.url}/api/v1/query', params=params)
        response.raise_for_status()
        return response.json()
    
    def query_range(self, query, start, end, step='15s'):
        """Range query"""
        params = {
            'query': query,
            'start': start,
            'end': end,
            'step': step
        }
        
        response = requests.get(f'{self.url}/api/v1/query_range', params=params)
        response.raise_for_status()
        return response.json()
    
    def get_pod_cpu_usage(self, pod_name, namespace='default', hours=1):
        """Get CPU usage for a pod over last N hours"""
        end = datetime.now()
        start = end - timedelta(hours=hours)
        
        query = f'sum(rate(container_cpu_usage_seconds_total{{pod="{pod_name}", namespace="{namespace}"}}[5m]))'
        
        result = self.query_range(
            query,
            start.timestamp(),
            end.timestamp(),
            step='30s'
        )
        
        return result['data']['result']

# Usage
prom = PrometheusClient()
cpu_data = prom.get_pod_cpu_usage('myapp-abc123', 'production')
for series in cpu_data:
    for value in series['values']:
        timestamp, usage = value
        print(f"{datetime.fromtimestamp(timestamp)}: {usage} cores")
```

### Example 4: Webhook Receiver (Flask)

```python
from flask import Flask, request, jsonify
import logging
import hmac
import hashlib

app = Flask(__name__)
logging.basicConfig(level=logging.INFO)

# GitHub webhook secret
WEBHOOK_SECRET = 'your-secret-here'

def verify_signature(payload, signature):
    """Verify GitHub webhook signature"""
    expected = hmac.new(
        WEBHOOK_SECRET.encode(),
        msg=payload,
        digestmod=hashlib.sha256
    ).hexdigest()
    
    return hmac.compare_digest(expected, signature)

@app.route('/webhook/github', methods=['POST'])
def github_webhook():
    # Verify signature
    signature = request.headers.get('X-Hub-Signature-256', '').replace('sha256=', '')
    if not verify_signature(request.data, signature):
        logging.warning("Invalid signature")
        return jsonify({'error': 'Invalid signature'}), 401
    
    event_type = request.headers.get('X-GitHub-Event')
    payload = request.json
    
    logging.info(f"Received {event_type} event")
    
    if event_type == 'push':
        repo = payload['repository']['full_name']
        branch = payload['ref'].split('/')[-1]
        commits = len(payload['commits'])
        logging.info(f"Push to {repo}:{branch} - {commits} commits")
        
        # Trigger CI/CD pipeline
        # trigger_build(repo, branch)
    
    elif event_type == 'pull_request':
        action = payload['action']
        pr_number = payload['number']
        pr_title = payload['pull_request']['title']
        logging.info(f"PR #{pr_number} {action}: {pr_title}")
    
    return jsonify({'status': 'ok'}), 200

if __name__ == '__main__':
    app.run(port=5000)
```

## Summary Table: requests Library

### HTTP Methods

| Method | requests function | Use |
|--------|-------------------|-----|
| GET | `requests.get(url)` | Fetch data |
| POST | `requests.post(url, json=data)` | Create data |
| PUT | `requests.put(url, json=data)` | Full update |
| PATCH | `requests.patch(url, json=data)` | Partial update |
| DELETE | `requests.delete(url)` | Delete data |

### Common Parameters

| Parameter | Purpose |
|-----------|---------|
| `params` | Query string parameters |
| `json` | JSON request body |
| `data` | Form data or raw body |
| `headers` | HTTP headers |
| `auth` | Authentication |
| `timeout` | Timeout in seconds |
| `verify` | SSL verification (True/False/cert path) |

### Response Methods

| Method | Returns |
|--------|---------|
| `response.json()` | Parsed JSON |
| `response.text` | Raw text |
| `response.content` | Raw bytes |
| `response.status_code` | HTTP status code |
| `response.ok` | True if 200-399 |
| `response.headers` | Response headers |
| `response.elapsed` | Response time |

### Exception Types

| Exception | When |
|-----------|------|
| `Timeout` | Request timed out |
| `ConnectionError` | Network problem |
| `HTTPError` | Bad status code (4xx, 5xx) |
| `RequestException` | Base class for all |
