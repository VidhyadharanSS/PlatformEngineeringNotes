### 9.3.1 Logging and Exception Handling: Professional Error Management

#### Why Logging and Error Handling Matter

`print()` statements are fine for debugging, but production code needs:
- **Log levels** – DEBUG, INFO, WARNING, ERROR, CRITICAL
- **Log destinations** – Console, files, syslog, external services
- **Structured logging** – JSON format for machine parsing
- **Proper error handling** – Graceful failure, not crashes

This note covers logging and exception handling. Note 9.3.2 covers HTTP clients; note 9.3.3 is the subchapter review.

**Backward references:** Bash exit codes from Module 3; error handling patterns from Module 3; Linux logging from Module 1 (syslog).

## Part 1: The `logging` Module – Professional Logging

### Basic Logging Setup

```python
import logging

# Basic configuration
logging.basicConfig(level=logging.INFO)

# Log messages
logging.debug("Debug message - detailed info")
logging.info("Info message - normal operation")
logging.warning("Warning message - something unexpected")
logging.error("Error message - something failed")
logging.critical("Critical message - application cannot continue")

# Output:
# INFO:root:Info message - normal operation
# WARNING:root:Warning message - something unexpected
# ERROR:root:Error message - something failed
# CRITICAL:root:Critical message - application cannot continue
```

### Log Levels

| Level | Numeric Value | When to Use |
|-------|---------------|-------------|
| `DEBUG` | 10 | Detailed debugging information |
| `INFO` | 20 | Confirmation that things are working |
| `WARNING` | 30 | Something unexpected, but application continues |
| `ERROR` | 40 | Something failed, application can continue |
| `CRITICAL` | 50 | Serious error, application cannot continue |

```python
import logging

# Set log level
logging.basicConfig(level=logging.WARNING)

# These won't appear (level < WARNING)
logging.debug("Debug")
logging.info("Info")

# These will appear
logging.warning("Warning")
logging.error("Error")
logging.critical("Critical")
```

### Logging to File

```python
import logging

# Configure file logging
logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s - %(name)s - %(levelname)s - %(message)s',
    filename='app.log',
    filemode='a'  # 'a' = append, 'w' = overwrite
)

logging.info("Application started")
logging.error("Something went wrong")

# Check log file
# tail app.log
# 2024-01-15 10:30:45,123 - root - INFO - Application started
# 2024-01-15 10:30:46,456 - root - ERROR - Something went wrong
```

### Formatting Log Messages

```python
import logging

# Format with timestamp, level, module, message
logging.basicConfig(
    format='%(asctime)s - %(levelname)s - %(module)s - %(message)s',
    datefmt='%Y-%m-%d %H:%M:%S',
    level=logging.INFO
)

logging.info("User logged in")
# 2024-01-15 10:30:45 - INFO - script - User logged in
```

### Available Format Variables

| Variable | Description |
|----------|-------------|
| `%(asctime)s` | Human-readable time |
| `%(created)f` | Time as float (since epoch) |
| `%(filename)s` | Filename |
| `%(funcName)s` | Function name |
| `%(levelname)s` | Log level name (INFO, ERROR) |
| `%(levelno)s` | Log level number (20, 40) |
| `%(lineno)d` | Line number |
| `%(message)s` | Log message |
| `%(module)s` | Module name |
| `%(name)s` | Logger name |
| `%(pathname)s` | Full path to file |
| `%(process)d` | Process ID |
| `%(thread)d` | Thread ID |

## Part 2: Advanced Logging Configuration

### Multiple Handlers (Console + File)

```python
import logging

# Create logger
logger = logging.getLogger('myapp')
logger.setLevel(logging.DEBUG)

# Console handler
console_handler = logging.StreamHandler()
console_handler.setLevel(logging.INFO)

# File handler
file_handler = logging.FileHandler('app.log')
file_handler.setLevel(logging.DEBUG)

# Formatter
formatter = logging.Formatter('%(asctime)s - %(name)s - %(levelname)s - %(message)s')
console_handler.setFormatter(formatter)
file_handler.setFormatter(formatter)

# Add handlers
logger.addHandler(console_handler)
logger.addHandler(file_handler)

# Use logger
logger.debug("This goes to file only (DEBUG)")
logger.info("This goes to both console and file")
logger.error("This goes to both")
```

### Rotating File Handler

```python
import logging
from logging.handlers import RotatingFileHandler

# Rotate after 5MB, keep 3 backups
handler = RotatingFileHandler(
    'app.log',
    maxBytes=5 * 1024 * 1024,  # 5MB
    backupCount=3
)

# Time-based rotation (daily)
from logging.handlers import TimedRotatingFileHandler
handler = TimedRotatingFileHandler(
    'app.log',
    when='midnight',
    interval=1,
    backupCount=30  # Keep 30 days
)

logger = logging.getLogger('myapp')
logger.addHandler(handler)
logger.setLevel(logging.INFO)
```

### JSON Logging (Structured Logging)

```python
import logging
import json
from datetime import datetime

class JSONFormatter(logging.Formatter):
    def format(self, record):
        log_entry = {
            'timestamp': datetime.utcnow().isoformat(),
            'level': record.levelname,
            'logger': record.name,
            'message': record.getMessage(),
            'module': record.module,
            'line': record.lineno
        }
        
        # Add exception info if present
        if record.exc_info:
            log_entry['exception'] = self.formatException(record.exc_info)
        
        return json.dumps(log_entry)

# Configure JSON logging
handler = logging.StreamHandler()
handler.setFormatter(JSONFormatter())
logger = logging.getLogger('myapp')
logger.addHandler(handler)
logger.setLevel(logging.INFO)

logger.info("User action", extra={'user_id': 123})  # Extra fields not automatically added
```

### Custom Logger with Extra Fields

```python
import logging
import json
from pythonjsonlogger import jsonlogger  # pip install python-json-logger

# Using python-json-logger library
handler = logging.StreamHandler()
formatter = jsonlogger.JsonFormatter(
    '%(asctime)s %(levelname)s %(name)s %(message)s %(user_id)s %(request_id)s'
)
handler.setFormatter(formatter)

logger = logging.getLogger('myapp')
logger.addHandler(handler)
logger.setLevel(logging.INFO)

# Add extra fields
logger.info("User logged in", extra={'user_id': 123, 'request_id': 'abc-123'})
# Output: {"asctime": "2024-01-15 10:30:45", "levelname": "INFO", "name": "myapp", "message": "User logged in", "user_id": 123, "request_id": "abc-123"}
```

## Part 3: Exception Handling with try/except

### Basic Exception Handling

```python
try:
    # Code that might fail
    number = int(input("Enter a number: "))
    result = 10 / number
    print(f"Result: {result}")
except ValueError:
    print("That's not a valid number!")
except ZeroDivisionError:
    print("Can't divide by zero!")
except Exception as e:
    print(f"Something else went wrong: {e}")
```

### Catching Specific Exceptions

```python
import os
import json

def load_config(filepath):
    try:
        with open(filepath, 'r') as f:
            return json.load(f)
    except FileNotFoundError:
        print(f"Config file not found: {filepath}")
        return {}
    except json.JSONDecodeError as e:
        print(f"Invalid JSON in config file: {e}")
        return {}
    except PermissionError:
        print(f"Permission denied reading: {filepath}")
        return {}

# Usage
config = load_config('/etc/myapp/config.json')
```

### try/except/else/finally

```python
try:
    file = open('data.txt', 'r')
    content = file.read()
except FileNotFoundError:
    print("File not found")
else:
    # Runs only if no exception occurred
    print(f"File content length: {len(content)}")
finally:
    # Always runs (for cleanup)
    if 'file' in locals():
        file.close()
    print("Cleanup complete")
```

### Raising Exceptions

```python
def validate_user(username, age):
    if not username:
        raise ValueError("Username cannot be empty")
    
    if age < 0:
        raise ValueError("Age cannot be negative")
    
    if age > 150:
        raise ValueError("Age seems unrealistic")
    
    return True

# Custom exception class
class ConfigurationError(Exception):
    """Raised when configuration is invalid"""
    pass

def load_config(config_path):
    if not config_path.exists():
        raise ConfigurationError(f"Config file not found: {config_path}")
    # ... rest of function
```

## Part 4: Logging Exceptions

### Logging with Traceback

```python
import logging

logging.basicConfig(level=logging.ERROR)

try:
    result = 10 / 0
except ZeroDivisionError:
    # Log with full traceback
    logging.exception("Division by zero occurred")
    # or
    logging.error("Division by zero occurred", exc_info=True)

# Output:
# ERROR:root:Division by zero occurred
# Traceback (most recent call last):
#   File "script.py", line 4, in <module>
#     result = 10 / 0
# ZeroDivisionError: division by zero
```

### Using `traceback` Module

```python
import logging
import traceback

try:
    risky_operation()
except Exception:
    error_string = traceback.format_exc()
    logging.error(f"Operation failed:\n{error_string}")
```

## Part 5: Practical Example – Production-Ready Script

```python
#!/usr/bin/env python3
"""
Production-ready script with logging and error handling
"""

import argparse
import logging
import sys
import os
from pathlib import Path

def setup_logging(log_level=logging.INFO, log_file=None):
    """Configure logging for the application"""
    handlers = []
    
    # Console handler
    console = logging.StreamHandler()
    console.setLevel(log_level)
    handlers.append(console)
    
    # File handler (if specified)
    if log_file:
        file_handler = logging.FileHandler(log_file)
        file_handler.setLevel(logging.DEBUG)
        handlers.append(file_handler)
    
    # Configure root logger
    logging.basicConfig(
        level=logging.DEBUG,  # Capture everything, handlers filter
        format='%(asctime)s - %(name)s - %(levelname)s - %(message)s',
        handlers=handlers
    )

class ConfigError(Exception):
    """Configuration error"""
    pass

class DeploymentError(Exception):
    """Deployment error"""
    pass

class Deployer:
    def __init__(self, environment='dev'):
        self.environment = environment
        self.logger = logging.getLogger(f'deployer.{environment}')
    
    def validate_config(self):
        """Validate configuration before deployment"""
        self.logger.info(f"Validating config for {self.environment}")
        
        required_vars = ['DB_HOST', 'DB_PASSWORD', 'API_KEY']
        missing = [var for var in required_vars if not os.environ.get(var)]
        
        if missing:
            raise ConfigError(f"Missing environment variables: {', '.join(missing)}")
        
        self.logger.info("Configuration validation passed")
        return True
    
    def deploy(self):
        """Perform deployment"""
        self.logger.info(f"Starting deployment to {self.environment}")
        
        try:
            self.validate_config()
            # Simulate deployment steps
            self.logger.info("Building Docker image...")
            self.logger.info("Pushing to registry...")
            self.logger.info("Updating Kubernetes manifests...")
            self.logger.info("Rolling out update...")
            
            # Simulate failure for demo
            if self.environment == 'prod':
                raise DeploymentError("Rollout timeout")
            
            self.logger.info(f"Deployment to {self.environment} completed successfully")
            return True
            
        except ConfigError as e:
            self.logger.error(f"Configuration error: {e}")
            return False
        except DeploymentError as e:
            self.logger.error(f"Deployment error: {e}")
            self.logger.exception("Full traceback:")
            return False
        except Exception as e:
            self.logger.critical(f"Unexpected error: {e}", exc_info=True)
            return False

def main():
    parser = argparse.ArgumentParser(description='Deployment tool')
    parser.add_argument('--env', choices=['dev', 'staging', 'prod'], default='dev')
    parser.add_argument('--log-level', choices=['DEBUG', 'INFO', 'WARNING', 'ERROR'], default='INFO')
    parser.add_argument('--log-file', help='Log file path')
    
    args = parser.parse_args()
    
    # Convert string level to logging constant
    log_level = getattr(logging, args.log_level)
    setup_logging(log_level, args.log_file)
    
    logger = logging.getLogger('main')
    logger.info(f"Starting deployment tool for {args.env}")
    
    deployer = Deployer(args.env)
    success = deployer.deploy()
    
    if success:
        logger.info("Deployment finished successfully")
        sys.exit(0)
    else:
        logger.error("Deployment failed")
        sys.exit(1)

if __name__ == '__main__':
    main()
```

## Summary Table: Logging

### Log Levels

| Level | Value | Use |
|-------|-------|-----|
| DEBUG | 10 | Detailed debugging |
| INFO | 20 | Normal operation |
| WARNING | 30 | Unexpected events |
| ERROR | 40 | Failures |
| CRITICAL | 50 | Fatal errors |

### Basic Configuration

| Parameter | Purpose |
|-----------|---------|
| `level` | Minimum log level |
| `format` | Log message format |
| `filename` | Log file path |
| `filemode` | 'a' (append) or 'w' (write) |

### Handlers

| Handler | Destination |
|---------|-------------|
| `StreamHandler` | Console (stdout/stderr) |
| `FileHandler` | Single file |
| `RotatingFileHandler` | Rotating files (size-based) |
| `TimedRotatingFileHandler` | Rotating files (time-based) |
| `SocketHandler` | Network socket |
| `SysLogHandler` | Syslog |

### Exception Handling

| Pattern | Use |
|---------|-----|
| `try/except` | Handle expected errors |
| `try/except/else` | Code that runs if no error |
| `try/finally` | Cleanup code |
| `try/except/else/finally` | All combined |
| `raise` | Raise exception |
| `raise ... from ...` | Exception chaining |
