### 9.2.1 Subprocess and Running Shell Commands: Bridging Python and the System

#### Why Subprocess Matters

Platform engineers need to run system commands from Python:
- Execute `kubectl` to manage Kubernetes clusters
- Run `docker` commands to build and push images
- Call `aws` or `gcloud` CLI tools
- Automate system administration tasks

The `subprocess` module replaces older methods like `os.system()` and `os.popen()` with a safer, more powerful interface.

This note covers subprocess. Note 9.2.2 covers arguments and path handling; note 9.2.3 is the subchapter review.

**Backward references:** Bash commands from Module 3 (running them from Python); Kubernetes from Module 5 (`kubectl` commands); Docker from Module 4 (`docker` commands).

---

## Part 1: Why Not `os.system()`?

The old way (`os.system()`) is dangerous and limited:

```python
import os

# DANGEROUS: Command injection vulnerability
user_input = "file.txt; rm -rf /"
os.system(f"ls {user_input}")  # This would delete files!

# Also: No easy way to capture output
# Also: No way to check return code properly
```

The `subprocess` module solves these problems.

---

## Part 2: `subprocess.run()` – The Modern Way

### Basic Usage

```python
import subprocess

# Run command, wait for completion
result = subprocess.run(['ls', '-la'])

# Check return code
if result.returncode == 0:
    print("Success!")
else:
    print(f"Failed with code {result.returncode}")
```

### Capturing Output

```python
import subprocess

# Capture stdout
result = subprocess.run(['ls', '-la'], capture_output=True, text=True)
print(result.stdout)  # Output as string
print(result.stderr)  # Error output

# Without text=True (returns bytes)
result = subprocess.run(['ls', '-la'], capture_output=True)
print(result.stdout.decode('utf-8'))
```

### Checking Return Code

```python
import subprocess

# Method 1: Check manually
result = subprocess.run(['ls', '/nonexistent'])
if result.returncode != 0:
    print("Command failed!")

# Method 2: check=True (raises exception on failure)
try:
    subprocess.run(['ls', '/nonexistent'], check=True)
except subprocess.CalledProcessError as e:
    print(f"Command failed with code {e.returncode}")
```

### Complete Example

```python
import subprocess

def run_command(cmd, capture=True, check=False):
    """Run a shell command and return output"""
    try:
        result = subprocess.run(
            cmd,
            capture_output=capture,
            text=True,
            check=check
        )
        return {
            'success': True,
            'stdout': result.stdout,
            'stderr': result.stderr,
            'returncode': result.returncode
        }
    except subprocess.CalledProcessError as e:
        return {
            'success': False,
            'stdout': e.stdout,
            'stderr': e.stderr,
            'returncode': e.returncode
        }

# Usage
result = run_command(['kubectl', 'get', 'pods'])
if result['success']:
    print(result['stdout'])
else:
    print(f"Error: {result['stderr']}")
```

## Part 3: Running Shell Commands with Pipe

### Using `shell=True` (Use with caution!)

```python
import subprocess

# Running shell built-ins or pipes
result = subprocess.run(
    "grep error /var/log/syslog | wc -l",
    shell=True,
    capture_output=True,
    text=True
)
print(f"Error count: {result.stdout.strip()}")

# WARNING: Command injection risk with user input!
user_input = "file.txt; rm -rf /"
# DANGEROUS: Never do this with user input!
result = subprocess.run(f"cat {user_input}", shell=True)
```

### Safer Alternative: Multiple subprocess calls

```python
import subprocess

# Instead of: cat file.txt | grep error | wc -l
# Do this:

# Step 1: cat file.txt
p1 = subprocess.Popen(['cat', 'file.txt'], stdout=subprocess.PIPE)

# Step 2: grep error
p2 = subprocess.Popen(['grep', 'error'], stdin=p1.stdout, stdout=subprocess.PIPE)
p1.stdout.close()

# Step 3: wc -l
p3 = subprocess.Popen(['wc', '-l'], stdin=p2.stdout, stdout=subprocess.PIPE)
p2.stdout.close()

# Get result
output = p3.communicate()[0].decode('utf-8')
print(f"Error lines: {output.strip()}")
```

## Part 4: `subprocess.Popen()` – Advanced Control

`Popen` gives fine-grained control over processes.

```python
import subprocess
import time

# Run process in background
process = subprocess.Popen(['ping', 'google.com'], stdout=subprocess.PIPE)

# Do other work while ping runs
time.sleep(5)

# Terminate the process
process.terminate()
process.wait()

# Read output
output = process.stdout.read().decode('utf-8')
print(output[:500])  # First 500 chars
```

### Real-time Output

```python
import subprocess

def run_with_output(cmd):
    """Run command and print output in real-time"""
    process = subprocess.Popen(
        cmd,
        stdout=subprocess.PIPE,
        stderr=subprocess.STDOUT,
        text=True,
        bufsize=1  # Line buffered
    )
    
    for line in process.stdout:
        print(line, end='')
    
    process.wait()
    return process.returncode

# Usage
run_with_output(['kubectl', 'apply', '-f', 'deployment.yaml'])
```

## Part 5: Practical Examples for Platform Engineers

### Example 1: Check Kubernetes Pod Status

```python
import subprocess
import json

def get_pod_status(namespace='default'):
    """Get status of all pods in a namespace"""
    cmd = ['kubectl', 'get', 'pods', '-n', namespace, '-o', 'json']
    
    result = subprocess.run(cmd, capture_output=True, text=True)
    
    if result.returncode != 0:
        print(f"Error: {result.stderr}")
        return []
    
    data = json.loads(result.stdout)
    pods = []
    
    for item in data.get('items', []):
        pod = {
            'name': item['metadata']['name'],
            'status': item['status']['phase'],
            'node': item['spec'].get('nodeName', 'N/A'),
            'restarts': sum(
                container.get('restartCount', 0)
                for container in item['status'].get('containerStatuses', [])
            )
        }
        pods.append(pod)
    
    return pods

# Usage
pods = get_pod_status('production')
for pod in pods:
    print(f"{pod['name']}: {pod['status']} (restarts: {pod['restarts']})")
```

### Example 2: Docker Image Management

```python
import subprocess
import re

def get_docker_images(repository=None):
    """Get list of Docker images"""
    cmd = ['docker', 'images', '--format', '{{.Repository}}:{{.Tag}}\t{{.Size}}']
    
    if repository:
        cmd.append(repository)
    
    result = subprocess.run(cmd, capture_output=True, text=True)
    
    if result.returncode != 0:
        return []
    
    images = []
    for line in result.stdout.strip().split('\n'):
        if not line:
            continue
        parts = line.split('\t')
        images.append({
            'name': parts[0],
            'size': parts[1] if len(parts) > 1 else 'unknown'
        })
    
    return images

def delete_old_images(days=30):
    """Delete images older than specified days"""
    cmd = ['docker', 'image', 'prune', '--all', '--filter', f'until={days}h', '--force']
    result = subprocess.run(cmd, capture_output=True, text=True)
    return result.returncode == 0
```

### Example 3: System Resource Monitoring

```python
import subprocess
import re

def get_cpu_usage():
    """Get CPU usage percentage"""
    cmd = ['top', '-bn1']
    result = subprocess.run(cmd, capture_output=True, text=True)
    
    # Parse top output for CPU line
    for line in result.stdout.split('\n'):
        if '%Cpu' in line:
            # Extract idle percentage
            match = re.search(r'(\d+\.\d+) id', line)
            if match:
                idle = float(match.group(1))
                return 100 - idle
    return None

def get_memory_usage():
    """Get memory usage percentage"""
    cmd = ['free', '-m']
    result = subprocess.run(cmd, capture_output=True, text=True)
    
    lines = result.stdout.split('\n')
    for line in lines:
        if line.startswith('Mem:'):
            parts = line.split()
            total = int(parts[1])
            used = int(parts[2])
            return (used / total) * 100
    return None

def get_disk_usage(path='/'):
    """Get disk usage percentage"""
    cmd = ['df', '-h', path]
    result = subprocess.run(cmd, capture_output=True, text=True)
    
    lines = result.stdout.split('\n')
    if len(lines) > 1:
        parts = lines[1].split()
        return parts[4]  # Use% column
    return None

def check_health():
    """Check system health"""
    cpu = get_cpu_usage()
    memory = get_memory_usage()
    disk = get_disk_usage()
    
    issues = []
    
    if cpu and cpu > 90:
        issues.append(f"High CPU usage: {cpu:.1f}%")
    if memory and memory > 90:
        issues.append(f"High memory usage: {memory:.1f}%")
    if disk and int(disk.rstrip('%')) > 90:
        issues.append(f"High disk usage: {disk}")
    
    if issues:
        print("WARNING: System health issues detected")
        for issue in issues:
            print(f"  - {issue}")
        return False
    else:
        print("System health OK")
        return True
```

## Part 6: Error Handling Best Practices

### Timeout Handling

```python
import subprocess

def run_with_timeout(cmd, timeout=30):
    """Run command with timeout"""
    try:
        result = subprocess.run(
            cmd,
            capture_output=True,
            text=True,
            timeout=timeout
        )
        return result.stdout, result.stderr, result.returncode
    except subprocess.TimeoutExpired:
        print(f"Command timed out after {timeout} seconds")
        return None, "Timeout", -1
    except subprocess.CalledProcessError as e:
        return e.stdout, e.stderr, e.returncode

# Usage
stdout, stderr, code = run_with_timeout(['kubectl', 'rollout', 'status', 'deployment/myapp'], timeout=60)
```

### Retry Logic

```python
import subprocess
import time

def run_with_retry(cmd, retries=3, delay=5, backoff=2):
    """Run command with retry on failure"""
    for attempt in range(retries):
        result = subprocess.run(cmd, capture_output=True, text=True)
        
        if result.returncode == 0:
            return result.stdout, result.stderr, result.returncode
        
        print(f"Attempt {attempt + 1} failed. Retrying in {delay}s...")
        time.sleep(delay)
        delay *= backoff
    
    return result.stdout, result.stderr, result.returncode

# Usage
stdout, stderr, code = run_with_retry(['kubectl', 'get', 'pods'], retries=5)
```

## Summary Table: subprocess Functions

| Function | Use Case | Returns |
|----------|----------|---------|
| `subprocess.run()` | Simple command, wait for completion | `CompletedProcess` |
| `subprocess.Popen()` | Advanced control (background, pipes) | `Popen` object |
| `subprocess.check_output()` | Get stdout only (legacy) | bytes |
| `subprocess.call()` | Run, get return code (legacy) | int |

### subprocess.run() Parameters

| Parameter | Purpose | Default |
|-----------|---------|---------|
| `capture_output` | Capture stdout/stderr | `False` |
| `text` | Return strings (not bytes) | `False` |
| `check` | Raise exception on failure | `False` |
| `timeout` | Timeout in seconds | `None` |
| `shell` | Use shell (caution!) | `False` |
| `env` | Custom environment | `None` |
| `cwd` | Working directory | `None` |

### Common Patterns

| Pattern | Code |
|---------|------|
| Run and ignore output | `subprocess.run(['cmd', 'arg'])` |
| Capture output | `result = subprocess.run(['cmd'], capture_output=True, text=True)` |
| Check success | `subprocess.run(['cmd'], check=True)` |
| With timeout | `subprocess.run(['cmd'], timeout=30)` |
| Custom env | `subprocess.run(['cmd'], env={'PATH': '/custom/bin'})` |
