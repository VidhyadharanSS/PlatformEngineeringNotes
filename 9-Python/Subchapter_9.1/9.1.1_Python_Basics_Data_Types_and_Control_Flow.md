### 9.1.1 Python Basics, Data Types, and Control Flow: Your First Python Scripts

#### Why Python for Platform Engineers

Bash is great for small scripts, but Python excels at:
- **Complex logic** – Conditionals, loops, data structures
- **Data processing** – JSON, YAML, CSV, APIs
- **Cross-platform** – Same code works on Linux, macOS, Windows
- **Rich ecosystem** – Thousands of libraries for any task
- **Maintainability** – Easier to read and test than complex Bash

This note covers Python basics. Note 9.1.2 covers file I/O and modules; note 9.1.3 is the subchapter review.

**Backward references:** Bash scripting from Module 3 (comparison with Python); Linux commands from Module 1 (we'll call them from Python later).

## Part 1: Python vs Bash – When to Use Which

| Task | Better Tool | Why |
|------|-------------|-----|
| One-liner file operations | Bash | Concise |
| Complex conditionals | Python | Readable |
| Data parsing (JSON, YAML) | Python | Built-in libraries |
| API calls | Python | `requests` library |
| System automation (5-10 lines) | Bash | Quick and simple |
| System automation (50+ lines) | Python | Maintainable |
| Cross-platform scripts | Python | Works everywhere |

## Part 5: Collections – Lists, Dictionaries, Sets, Tuples

### Lists (Ordered, Mutable)

```python
# Create list
fruits = ["apple", "banana", "orange"]
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]

# Access elements (0-indexed)
print(fruits[0])    # apple
print(fruits[-1])   # orange (last element)

# Slicing
print(fruits[0:2])  # ['apple', 'banana']
print(fruits[:2])   # ['apple', 'banana']
print(fruits[1:])   # ['banana', 'orange']

# Modify
fruits.append("grape")        # Add to end
fruits.insert(1, "blueberry") # Insert at index
fruits.remove("banana")       # Remove by value
popped = fruits.pop()         # Remove and return last
fruits[0] = "cherry"          # Replace

# Check if contains
if "apple" in fruits:
    print("Found!")

# Length
len(fruits)
```

### List Comprehensions (Pythonic!)

```python
# Traditional loop
squares = []
for i in range(10):
    squares.append(i * i)

# List comprehension (more Pythonic)
squares = [i * i for i in range(10)]

# With condition
even_squares = [i * i for i in range(10) if i % 2 == 0]

# Nested comprehension
pairs = [(x, y) for x in [1, 2, 3] for y in ['a', 'b', 'c']]
```

### Dictionaries (Key-Value Pairs, Mutable)

```python
# Create dictionary
user = {
    "name": "Alice",
    "age": 30,
    "email": "alice@example.com"
}

# Access values
print(user["name"])        # Alice
print(user.get("phone"))   # None (instead of error)

# Modify
user["age"] = 31
user["phone"] = "555-1234"

# Remove
del user["email"]
phone = user.pop("phone")

# Check if key exists
if "name" in user:
    print(user["name"])

# Iterate
for key in user:
    print(key, user[key])

for key, value in user.items():
    print(f"{key}: {value}")

for value in user.values():
    print(value)
```

### Dictionary Comprehensions

```python
# Square numbers as values
squares = {i: i * i for i in range(5)}
# {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# Filtering
even_squares = {i: i * i for i in range(10) if i % 2 == 0}
```

### Sets (Unordered, Unique, Mutable)

```python
# Create set
unique_numbers = {1, 2, 3, 3, 4}  # {1, 2, 3, 4}
unique_list = set([1, 2, 2, 3])    # {1, 2, 3}

# Set operations
set_a = {1, 2, 3, 4}
set_b = {3, 4, 5, 6}

print(set_a | set_b)  # Union: {1, 2, 3, 4, 5, 6}
print(set_a & set_b)  # Intersection: {3, 4}
print(set_a - set_b)  # Difference: {1, 2}
print(set_a ^ set_b)  # Symmetric difference: {1, 2, 5, 6}

# Modify
set_a.add(7)
set_a.remove(1)       # Error if not exists
set_a.discard(1)      # No error if not exists
```

### Tuples (Ordered, Immutable)

```python
# Create tuple
coordinates = (10, 20)
rgb = (255, 0, 255)
single_item = (5,)  # Note the comma!

# Access (like lists)
print(coordinates[0])  # 10

# Can't modify (immutable)
# coordinates[0] = 15  # TypeError!

# Useful for returning multiple values from function
def get_user():
    return "Alice", 30, "alice@example.com"

name, age, email = get_user()  # Tuple unpacking
```

## Part 6: Control Flow

### If/Elif/Else

```python
age = 25

if age < 18:
    print("Minor")
elif age < 65:
    print("Adult")
else:
    print("Senior")

# Multiple conditions
if age >= 18 and age < 65:
    print("Working age")

if age < 18 or age > 65:
    print("Not working age")

# Ternary operator (inline if)
status = "Adult" if age >= 18 else "Minor"
```

### For Loops

```python
# Loop over list
fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(fruit)

# Loop with index
for i, fruit in enumerate(fruits):
    print(f"{i}: {fruit}")

# Loop over range
for i in range(5):      # 0, 1, 2, 3, 4
    print(i)

for i in range(2, 5):   # 2, 3, 4
    print(i)

for i in range(0, 10, 2):  # 0, 2, 4, 6, 8
    print(i)

# Loop over dictionary
user = {"name": "Alice", "age": 30}
for key, value in user.items():
    print(f"{key}: {value}")
```

### While Loops

```python
count = 0
while count < 5:
    print(count)
    count += 1

# Infinite loop (break to exit)
while True:
    response = input("Type 'quit' to exit: ")
    if response == "quit":
        break

# Continue (skip iteration)
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)  # Only odd numbers
```

## Part 7: Functions

### Basic Functions

```python
def greet():
    """Simple function with no parameters"""
    print("Hello!")

def greet_person(name):
    """Function with one parameter"""
    print(f"Hello, {name}!")

def add(a, b):
    """Function with return value"""
    return a + b

# Call functions
greet()
greet_person("Alice")
result = add(5, 3)  # 8
```

### Default Arguments

```python
def greet(name="World"):
    print(f"Hello, {name}!")

greet()           # Hello, World!
greet("Alice")    # Hello, Alice!
```

### Keyword Arguments

```python
def create_user(name, age, email):
    print(f"Name: {name}, Age: {age}, Email: {email}")

# Positional arguments
create_user("Alice", 30, "alice@example.com")

# Keyword arguments (order doesn't matter)
create_user(email="alice@example.com", name="Alice", age=30)
```

### Variable Arguments (*args, **kwargs)

```python
# *args - variable number of positional arguments
def sum_all(*args):
    return sum(args)

print(sum_all(1, 2, 3))      # 6
print(sum_all(1, 2, 3, 4, 5)) # 15

# **kwargs - variable number of keyword arguments
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="Alice", age=30, city="New York")
```

### Return Values

```python
def divide(a, b):
    if b == 0:
        return None  # Indicates error
    return a / b

def get_user_stats():
    return "Alice", 30, "alice@example.com"

# Multiple return values (tuple unpacking)
name, age, email = get_user_stats()
```

### Lambda Functions (Anonymous)

```python
# Simple lambda
square = lambda x: x * x
print(square(5))  # 25

# Used with sort, filter, map
numbers = [1, 2, 3, 4, 5]
squared = list(map(lambda x: x * x, numbers))
evens = list(filter(lambda x: x % 2 == 0, numbers))
```

## Part 8: Practical Examples for Platform Engineers

### Example 1: Parse Log File

```python
#!/usr/bin/env python3
import re

def parse_log_line(line):
    """Extract timestamp, level, and message from log line"""
    pattern = r'(\d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2}) (\w+) (.*)'
    match = re.match(pattern, line)
    if match:
        return {
            'timestamp': match.group(1),
            'level': match.group(2),
            'message': match.group(3)
        }
    return None

log_line = "2024-01-15 10:30:45 ERROR Failed to connect to database"
parsed = parse_log_line(log_line)
print(parsed)
```

### Example 2: Count Errors by Type

```python
#!/usr/bin/env python3
from collections import defaultdict

def count_errors(log_file):
    error_counts = defaultdict(int)
    
    with open(log_file, 'r') as f:
        for line in f:
            if 'ERROR' in line:
                # Extract error type (simple example)
                if 'Database' in line:
                    error_counts['database'] += 1
                elif 'Network' in line:
                    error_counts['network'] += 1
                else:
                    error_counts['other'] += 1
    
    return dict(error_counts)

# Usage
# errors = count_errors('/var/log/myapp.log')
# print(errors)
```

### Example 3: Check Disk Usage

```python
#!/usr/bin/env python3
import os
import shutil

def check_disk_usage(path='/', threshold=90):
    """Check if disk usage exceeds threshold percentage"""
    usage = shutil.disk_usage(path)
    percent = (usage.used / usage.total) * 100
    
    if percent > threshold:
        print(f"WARNING: Disk usage at {percent:.1f}%")
        return False
    else:
        print(f"OK: Disk usage at {percent:.1f}%")
        return True

check_disk_usage('/')
```
